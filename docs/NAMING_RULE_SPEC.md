# CutLab 命名与归档模板语法

> 版本：v1.0 · 2026-06-25

---

## 命名模板（Naming Template）

用于生成规范文件名，例如：

```
EP{EP:02}_S{SC:02}_C{CUT:03}_{TYPE}
→ EP01_S02_C003_原画.png
```

### 占位符

| 占位符 | 含义 | 示例输出 |
|--------|------|----------|
| `{EP:02}` | 集数，2 位补零 | `01` |
| `{SC:02}` | 场号，2 位补零 | `02` |
| `{CUT:03}` | 镜号，3 位补零 | `003` |
| `{TYPE}` | 资产类型后缀 | `原画` |

### 规则

- 模板 **必须包含** `{CUT` 片段（如 `{CUT:03}`）
- 前缀文字可自由组合，例如 `EP{EP:02}`、`C{CUT:03}`
- 类型后缀在项目中默认映射：

| 类型 | 后缀 |
|------|------|
| Storyboard | 分镜 |
| Keyframe | 原画 |
| Inbetween | 动画 |
| Background | 背景 |
| Render | 渲染 |

---

## 归档路径模板（Archive Path Pattern）

用于生成目录结构，例如：

```
EP{EP:02}/S{SC:02}/C{CUT:03}/{TYPE}
→ EP01/S02/C003/原画/
```

占位符与命名模板相同。归档根路径（项目设置中的「归档根路径」）为所有相对路径的基准。

---

## 内置文件识别规则

扫描时按以下顺序匹配文件名（不含扩展名）：

| 模式 | 示例 |
|------|------|
| 标准规范名 | `EP01_S02_C001_原画` |
| 简单卡号 | `C003_分镜` |
| 中文卡号 | `1卡原画` |
| 序号卡号（无类型） | `第3卡` → 识别卡号但标记未识别类型 |

---

## 模板示例

### 电视动画标准

- 命名：`EP{EP:02}_S{SC:02}_C{CUT:03}_{TYPE}`
- 归档：`EP{EP:02}/S{SC:02}/C{CUT:03}/{TYPE}`

### 简单单集

- 命名：`C{CUT:03}_{TYPE}`
- 归档：`C{CUT:03}/{TYPE}`
